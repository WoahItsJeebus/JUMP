# <font color="#00b7ffff" face="Consolas">JUMP — Jeebus' Unoptimized Minesweeper Player</font>

<font color="#00ffbfff" face="Consolas">JUMP is a playable Minesweeper clone in AutoHotkey v2.0.19 — built for practice, not automation.
Classic controls, first-click safe, timer + mine counter, standard difficulties, and optional Custom boards.

### Dependencies
- [AutoHotKey v2.0.19+](https://autohotkey.com/download/ahk-v2.exe)
